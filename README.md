# Sanayi Marketi

A web platform connecting users searching for materials with companies that manufacture or sell those materials.

## Tech Stack

### Server
- Java 21
- Spring Boot 4
- PostgreSQL

### Client
- React
- Vite

## Folder Structure

```
├── server/    # Spring Boot REST API
├── client/    # React SPA with Vite
└── README.md
```
